# AI工具导航

收录 130+ 免费 AI 工具和办公资源，分类齐全，持续更新。

## 功能

- 13 个分类：AI聊天、AI写作、AI图像、AI视频、AI办公、AI编程、AI设计、AI智能体、办公效率、开发工具、优惠购物、学习资源、设计资源
- 两级页面：首页分类导航 → 分类详情页工具列表
- 全局搜索：按名称、标签、描述搜索
- 深色/浅色主题切换
- 响应式设计，手机和电脑都好用
- 管理后台：浏览器打开 admin.html 即可管理内容
- 所有工具链接可直接点击跳转（新标签页打开）

## 本地使用

直接用浏览器打开 `index.html` 即可。

## 部署到 GitHub Pages

1. 将此仓库推送到 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 Deploy from branch
4. Branch 选择 main，文件夹选 /(root)
5. 保存后等待 1-2 分钟，即可通过 `https://你的用户名.github.io/仓库名/` 访问

## 管理后台

打开 `admin.html` 可以：
- 添加、编辑、删除分类和工具
- 导出 JSON 数据
- 一键生成新的 index.html
